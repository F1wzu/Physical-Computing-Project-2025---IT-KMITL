# Physical-Computing-Project-2025 - IT-KMITL
# COLOR MEMORY GAME
# บทคัดย่อ
   “Color Memory Game” เป็นโครงงานการสร้างเกมด้วยไมโครคอนโทรลเลอร์ (Microcontroller) ซึ่งเป็นส่วนหนึ่งของรายวิชา “การประมวลผลทางกายภาพ (Physical Computing)” โดยมีวัตถุประสงค์เพื่อพัฒนาทักษะและประยุกต์ใช้ความรู้ที่ได้รับจากรายวิชานี้โดยการสร้างผลงานจริง
   เกมดังกล่าวออกแบบมาเพื่อสร้างความบันเทิงและท้าทายความจำของผู้เล่น มีการไต่ระดับความยาก เสียงประกอบจาก Passive Buzzer Module เพื่อเพิ่มอรรถรสในการเล่น รวมถึงการแสดงผลต่าง ๆ ผ่านจอ LCD (Liquid Crystal Display) เพื่อระบุช่วงการจำ (Memorize) ช่วงการเล่น ระดับความยากง่าย (Level) คะแนนของปัจจุบัน และคะแนนสูงสุด (High Score) 
   โครงงานนนี้จึงเป็นสื่อการเรียนรู้ที่ช่วยเสริมทักษะทางด้านการเขียนโปรแกรมและการประมวลผลทางกายภาพได้อย่างมีประสิทธิภาพ
